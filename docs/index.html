Quick Installation

Get started with Oracle Claude Code Skills in seconds

1

Add the marketplace

# In Claude Code terminal
/plugin marketplace add frankxai/claude-code-oracle-skills
2

Install all plugins

/plugin install oracle-adk oracle-agent-spec oci-services-expert oracle-ai-architect oracle-diagram-generator
3

Start using slash commands

# Generate a RAG architecture diagram
/oci-diagram rag mermaid

# Scaffold an ADK agent
/adk-agent customer-support multi-agent

# Estimate OCI costs
/oci-cost "RAG platform with vector search"

Or install manually with the install script:

curl -fsSL https://raw.githubusercontent.com/FrankXAI/claude-code-oracle-skills/main/install.sh | bash
