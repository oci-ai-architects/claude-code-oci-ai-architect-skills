**Think Like an AI Architect. Visualize Like Oracle.**

- **Research-Grounded** - Validated against Oracle documentation

## Visual Style Matrix for AI Architects

| Style | Best For | Audience | Characteristics |
|-------|----------|----------|-----------------|
| **executive** | Board presentations, proposals | C-Suite, Business | Clean, minimal, outcome-focused, large text, 3-4 layers max |
| **technical** | Architecture reviews, design docs | Architects, Engineers | Dense detail, all services shown, data flows, specs |
| **futuristic** | Innovation showcases, demos | Mixed, Tech-forward | Dark background, neon accents, AI/agent emphasis |
| **isometric** | Solution overviews, marketing | Sales, Marketing, Prospects | 3D depth, visual appeal, balanced detail |
| **blueprint** | Implementation guides, RFPs | Technical Decision Makers | Precise, annotation-heavy, specification-focused |

### Recommended Default: `executive` for customer-facing, `technical` for internal

ORACLE BRANDING:
- Oracle Red (#C74634) for headers and key services
